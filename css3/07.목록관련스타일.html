목록 관련 스타일(list~)

list-style-type : 블릿기호를 변경시켜줄 때 사용

list-style-image : 블릿기호를 이미지로 적용할 때 사용

list-style-pisition : 블릿기호의 위치를 조정할 때 사용

        * 순서 없는 목록(ul)
        선택자 {
            list-style-type: disc | circle | square | none;
        }

        * 순서 있는 목록(ol)
        선택자 {
            list-style-type: decimal | decimal-leading-zero;
            list-style-type: lower-alpha | upper-alpha;
            list-style-type: lower-roman : upper-roman;
        }

순서 없는 list

  • HTML5
  • CSS3
  • JAVASCRIPT
  • JQUERY

순서 있는 list

  1. HTML5
  2. CSS3
  3. JAVASCRIPT
